Wearables
in the Dark

Making a luminous brooch (or accessory) using textile waste and flexible conductive material. The brooch has a light sensor that allows the LED to light up when it's dark.The workshop is open to the general public, with no prior knowledge of electronics or textiles required.
Let's Start Learn more about NANA
This is a 2h meeting to discuss the wearables mind set and have some experimentation
I like to think that design is a framework. Actually, I love build frameworks for myself. NANA was born with this passion, create a creative and technical enviroment for developing ideais in soft circuits. So, let's see what happening when we put minds and materials to think together.
